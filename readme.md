# SaveMyWindows

A tool that restores your windows after a change of displays, instead of having to manually rearrange them again.

---

## Installation

```bash
pip install -r requirements.txt
```

---

## Usage

```bash
python main.py
```

---

## Parameters

| Name     | Type  | Description |
|----------|-------|-------------|
| `--name` | `str` | Your name.  |
